# Muscle Memory

Muscle Memory is a code-only Risk of Rain 2 mod with run-scoped, logarithmic mastery for Primary/Secondary/Utility/Special slots.

## Features
- Skills level forever via `floor(log2(1 + P / K))` — no cap.
- **12 milestone perks** across 3 tiers per skill slot (crit, bleed, attack speed, CDR, stock refund, armor, barrier, cooldown refund, and more).
- **Color-coded HUD** with progress bars above each skill icon.
- Optional Cold Start penalties (enabled by default).
- Optional proficiency decay for idle slots (off by default).
- Survivor scaling multiplier for per-profile tuning.
- Chat spam control — broadcast only milestones or all level-ups.
- Host-authoritative progression with replicated slot levels for multiplayer consistency.
- Late-joining clients receive a full snapshot immediately.
